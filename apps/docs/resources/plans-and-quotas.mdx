---

## Règles de quota

- les messages et les statuts partagent le même quota mensuel
- `effectiveOutboundUsage = messagesCount + statusesCount`
- le quota API mensuel est distinct du rate limit temps réel
- pour les messages planifiés, le quota est revérifié au moment de l'envoi effectif

## Suivre votre consommation

### Endpoint historique compatible

```bash
curl https://srv.msgflash.com/api/v1/usage \
  -H "x-api-key: msgf_live_your_api_key_here"
```

### Endpoint billing enrichi

```bash
curl https://srv.msgflash.com/api/v1/billing/usage \
  -H "x-api-key: msgf_live_your_api_key_here"
```
