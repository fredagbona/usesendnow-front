---
title: GET Historique des paiements
description: Liste paginée des paiements du compte.
---

## Endpoint

```txt
GET /api/v1/billing/payments
```

## Paramètres query

| Nom | Type | Requis | Description |
|---|---|---|---|
| `page` | integer | non | Défaut `1` |
| `limit` | integer | non | Défaut `20`, max `100` |
