---
title: PUT Mettre à jour un template
description: Met à jour un template existant et recalcule ses variables.
---

## Endpoint

```txt
PUT /api/v1/templates/{id}
```

## Paramètres path

| Nom | Type | Requis | Description |
|---|---|---|---|
| `id` | UUID | oui | ID du template |

## Body

| Champ | Type | Requis | Description |
|---|---|---|---|
| `name` | string | non | Nouveau nom |
| `body` | string | non | Nouveau body |
| `mediaUrl` | string | non | Nouvelle URL media |

## Réponse succès `200`

```json
{
  "data": {
    "id": "tmpl_uuid",
    "variables": ["contact.firstName"]
  }
}
```
