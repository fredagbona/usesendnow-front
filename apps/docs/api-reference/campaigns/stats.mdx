---
title: GET Statistiques d'une campagne
description: Récupère les statistiques et la timeline d'exécution d'une campagne.
---

## Endpoint

```txt
GET /api/v1/campaigns/{id}/stats
```

## Paramètres path

| Nom | Type | Requis | Description |
|---|---|---|---|
| `id` | UUID | oui | ID de la campagne |
